# מסמך אפיון ופרומפט בנייה — דף נחיתה: Harel Lesnick

## 1. סוג האתר
דף נחיתה (Landing Page) חד-עמודי, פורטפוליו אישי למהנדס סאונד / צלם / עורך וידאו. לא אפליקציה — אין משתמשים, התחברות או מסד נתונים.

## 2. מטרת האתר
להציג את השירותים והעבודות של הראל לסניק ולהוביל לקוחות פוטנציאליים לקביעת פגישה/שיחה איתו.

## 3. קהל היעד
לקוחות פוטנציאליים: אומנים ולהקות שצריכים הקלטה/מיקס/מאסטרינג, מפתחי משחקי מחשב שצריכים סאונד דיזיין, ויוצרי תוכן שצריכים עריכת וידאו/צילום.

## 4. שפה וכיוון האתר
דו-לשוני: עברית (RTL) כברירת מחדל + אנגלית (LTR), עם מתג שפה בתפריט שמשנה גם את כיוון הדף (`dir="rtl"`/`dir="ltr"`) וגם את כל התוכן.
